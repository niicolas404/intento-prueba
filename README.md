prompt 1

usa #file:index.html y #file:style.css para crear la estructura de una tienda destinada al coemrcio de casas. Necesito que cada producto cuente sea presentado como una tarjeta (en donde irá la imagen de la casa en toda la cara) Abajo de esta tarjeta debe ir la descripción de la casa, lo que ofrece y el precio. Debe haber un encabezado que diga Mundo hogar. Abajo de ello debe haber un menú que contenga "inicio, casas, cuenta, sobre nosotros". Es importante que USES SOLAMENTE ESTOS DOS LENGUAJES DE PROGRAMACIÓN. No te salgas de lo que es HTML y CSS. Puedes termianr el codigo con un footer que tenga una lista con imagenes de las redes sociales.

En cuanto al css, aplica un estilo minimalista, usa colores sobrios que denoten profesionalismo. Además, asegurate de que absolutamente todo sea responsive. Y que cada imagen que se pueda agregar no se recorte por el tamaño, sino que se ajuste correctamente en la tarjeta


resultado: se logró una estructura bastante básica de lo que sería una tienda. Aún faltan cosas que se pueden pulir.
